# 1.新时代坚持和发展中国特色社会主义
 - 时间顺序：
1. 什么是社会主义、怎样建设社会主义
    ——主要对应**邓小平理论**
2. 建设什么样的党、怎样建设党
    ——主要对应 **三个代表重要思想**，强调党的建设问题。
3. 新形势下需要什么样的发展、怎样发展
    ——主要对应**科学发展观**，重点是发展方式、以人为本、全面协调可持续。
4. 新时代坚持和发展什么样的中国特色社会主义、怎样坚持和发展中国特色社会主义
    ——对应**习近平新时代中国特色社会主义思想**，这是进入新时代后的重大课题。

 - 无产阶级专政不是中国特色，人民民主专政才是
 - 新时代：19大，全面建设社会主义现代化强国；社会主要矛盾变了、党的任务变了、中国与世界关系变了
 - 新时代全社会研发经费支出全球**第二**，谷物总产量世界**第一**
 - 概念辨析
	 - 基本路线：一个中心两个基本点(以经济建设为中心，坚持四项基本原则，坚持改革开放），是党和国家生命线、人民幸福线
	 - 思想路线：实事求是

- 习思想回答的**重大时代课题**：中国特色社会主义、社会主义现代化强国、马克思主义政党，**不包括实现什么样的发展、怎样发展**（这是科学发展观的内容）
- 习思想的**世界观和方法论**：六个坚持（人民、自信、守正、问题、系统、天下），但是**没有 “坚持党的领导，坚持实事求是”**
- 中国的问题：从中国国情出发，由中国人自己解答，**不能首先看世界情况**
	错题：


# 其他
 - 全面建成小康社会2020年完成，2021年公布，和新时代（十八大/十九大）不重合
 - 我国的根本政治制度是**人民代表大会制度**，**而非党的领导制度**
 - 我国的基本政治制度是中国共产党领导的多党合作和政治协商制度、民族区域自治制度、基层群众自治制度
 - 改革开放是决定当代中国命运的**关键一招**，而非**关键一步**
 - 国家安全是**民族复兴的根基**，社会稳定是**国家强盛的前提**。和高质量发展无关
 - 国有企业是政治基础
 - 高标准市场体系不含高效规范
