| **异常类型** | **继承自** | **描述** |
| ------------------------------------- | -------------------- | --------------------------------------- |
| **逻辑错误（Logic Errors）** | | |
| `std::logic_error` | `std::exception` | 表示程序中的逻辑问题，是逻辑错误的基类。 |
| `std::invalid_argument` | `std::logic_error` | 表示传递给函数的参数无效。 |
| `std::domain_error` | `std::logic_error` | 表示输入值超出了函数的定义域（如数学函数的无效输入）。 |
| `std::length_error` | `std::logic_error` | 表示尝试创建超出允许大小的对象时抛出的异常。 |
| `std::out_of_range` | `std::logic_error` | 表示访问容器时索引超出范围时抛出的异常。 |
| **运行时错误（Runtime Errors）** | | |
| `std::runtime_error` | `std::exception` | 表示程序运行时由于外部条件而发生的错误，是运行时错误的基类。 |
| `std::range_error` | `std::runtime_error` | 表示结果超出有效范围时抛出的异常。 |
| `std::overflow_error` | `std::runtime_error` | 表示算术运算导致溢出时抛出的异常。 |
| `std::underflow_error` | `std::runtime_error` | 表示算术运算导致下溢时抛出的异常。 |
| **其他标准异常（Other Standard Exceptions）** | | |
| `std::bad_alloc` | `std::exception` | 表示内存分配失败时抛出的异常（如 `new` 操作符分配内存失败）。 |
| `std::bad_cast` | `std::exception` | 表示类型转换失败时抛出的异常（如 `dynamic_cast` 的非法转换）。 |
| `std::bad_typeid` | `std::exception` | 表示访问空类型信息时抛出的异常（如对空指针使用 `typeid`）。 |
| `std::bad_function_call` | `std::exception` | 表示调用了未绑定的 `std::function` 对象时抛出的异常。 |
| `std::bad_weak_ptr` | `std::exception` | 表示尝试访问已销毁的 `std::weak_ptr` 时抛出的异常。 |
