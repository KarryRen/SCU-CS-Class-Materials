| **类别** | **异常类** | **描述** |
| ---------- | ----------------------- | -------------- |
| **基类** | `std::exception` | 所有标准异常的基类 |
| **逻辑错误** | `std::logic_error` | 逻辑错误基类 |
| | `std::invalid_argument` | 无效参数 |
| | `std::domain_error` | 数学定义域错误 |
| | `std::length_error` | 超出对象允许长度 |
| | `std::out_of_range` | 容器访问越界 |
| **运行时错误** | `std::runtime_error` | 运行时错误基类 |
| | `std::overflow_error` | 算术运算溢出 |
| | `std::underflow_error` | 算术运算下溢 |
| | `std::range_error` | 结果超出范围 |
| **内存分配相关** | `std::bad_alloc` | 内存分配失败 |
| **类型相关** | `std::bad_cast` | 类型转换失败 |
| | `std::bad_typeid` | 空指针调用 `typeid` |

## 重点识记

`runtime_error`：运行时错误，常见于**算数过程**（不是数据越界！）
`logic_error`：逻辑错误，遇到不能访问的东西和错误的参数
* `invalid_argument`：错误的参数
* `out_of_range`：**访问数组越界**
* `length_error`：超过对象允许的长度

`exception` ：所有异常的基类，一般放在倒数第二的位置
`...`：表示任意异常，一般放在最后

```cpp
#include <iostream>
#include <stdexcept> // 包含标准异常类
using namespace std;

void testException(int code) {
 if (code == 1) {
 throw runtime_error("Runtime error occurred!");
 } else if (code == 2) {
 throw invalid_argument("Invalid argument provided!");
 } else if (code == 3) {
 throw out_of_range("Out of range error!");
 } else {
 throw "Unknown exception!";
 }
}

int main() {
 try {
 testException(2); // 这里可以修改参数以触发不同的异常
 } catch (const runtime_error& e) {
 cout << "Caught a runtime_error: " << e.what() << endl;
 } catch (const invalid_argument& e) {
 cout << "Caught an invalid_argument: " << e.what() << endl;
 } catch (const out_of_range& e) {
 cout << "Caught an out_of_range: " << e.what() << endl;
 } catch (const char* e) {
 cout << "Caught an unknown exception: " << e << endl;
 } catch (...) {
 cout << "Caught an unspecified exception!" << endl;
 }

 cout << "Program continues running..." << endl;

 return 0;
}

```
