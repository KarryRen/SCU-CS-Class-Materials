## 常变量

**常变量**声明时必须定义数值，且之后不能修改
```cpp
const int a = 5; // 常变量a
```
## 常成员
### 常数据成员

用 `const` 修饰的数据成员必须**初始化列表**来初始化，通常不能修改值。

如果**强制修改 `const` 成员**，需要使用：
```cpp
const int memory = 10;
const_cast<int&>memory = 20;
```
这里的 `const_cast<int&>` 强制将 `int` 类型的值修改到 20
> 这不是推荐的做法，除非题目特别要求，不然不要这样做!
### 常函数成员

在函数的声明和实现之间，加入 `const`，就得到常函数成员。这种函数不能修改任何成员变量的值

```cpp
#include <iostream>

class MyClass {
 const int x; // 常数据成员
public:
 MyClass(int value) : x(value) {} // 必须通过初始化列表初始化
 void print() const {
 std::cout << "x = " << x << std::endl;
 }
};

int main() {
 MyClass obj(10);
 obj.print();
 // obj.x = 20; // 错误，x 是常数据成员，不能修改
 return 0;
}
```
### 常对象

声明完对象后，新建对象前加上 `const` 关键字得到常对象：
* 只读对象，不能修改任何值
* **只能调常成员函数**

```cpp
#include <iostream>

class MyClass {
 int x;
public:
 MyClass(int value) : x(value) {}

 void setValue(int value) {
 x = value; // 非常函数
 }

 int getValue() const { // 常函数
 return x;
 }
};

int main() {
 const MyClass obj(10); // 常对象
 // obj.setValue(20); // 错误，常对象不能调用非常函数
 std::cout << obj.getValue() << std::endl; // 常对象只能调用常函数
 return 0;
}

```
