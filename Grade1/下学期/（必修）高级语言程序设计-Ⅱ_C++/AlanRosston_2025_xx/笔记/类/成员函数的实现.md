函数的**声明**可以：
1. 不指定参数名，只声明有哪些参数
2. 不需要函数体 `{}`

函数的**实现**需要：
1. 对于类的成员函数，需要加上函数的返回类型，如 `void Person::display() {...}`
2. 对于类的构造函数等特殊函数，不需要加上函数类型，如 `Person::Person() {}`

示例：
```cpp
#include <iostream>
using namespace std;

class Person {
private:
int age;
string name;

public:
	Person();
	Person(int); // 函数重载
	Person(int, string);

	void display() const; // 声明一个常函数
};

// 类外实现
Person::Person() {
name = "null";
age = 0;
cout << "Void params constructor" << endl;
}

Person::Person(int age) {
this->age = age;
name = "null";
cout << "One param constructor" << endl;
}

Person::Person(int age, string name) {
this->age = age;
this->name = name;
cout << "Two params constructor" << endl;
}

void Person::display() const {
cout << this->name << " is " << this->age << endl;
}

int main(void)
{
Person p0;
Person p1(10);
Person p2(20, "AlanRosston");

p0.display();
p1.display();
p2.display();

return 0;
}
```
