## 友元关系的特征

在 C++ 中，**友元关系（friendship）** 是一种允许一个类的**非成员函数**或**其他类**访问其私有成员（`private`）和保护成员（`protected`）的机制

声明**友元关系**后：
* 当前类的 `private` 和 `protected` 内的参数可以访问了
* 通常，友元函数的**参数列表**包含该类的指针或者引用，因为需要访问其 `private` 或 `protected` 成员；特别地，用于修改**静态成员**的友元函数可以不包括上述
```cpp
class Building {
	private:
		static int id;
	public:
		friend void set_id(int x); //类内声明
		...
}

//类外实现
void set_id(int x) {
	Building::id = x
}
```
* 友元会**破坏封装**，常用于运算符重载；友元关系不必放在 `public` 下，在哪都一样
* 使用**友元类**时，通常需要在该类前实现声明友元类，如 `class Printer`，否则编译器不知道这是友元类
* **友元函数**通常类内声明，类外实现；特别地，若类内声明的是**模板友元函数**，必须在类外重新实现一次

## 友元类示例
```cpp
#include <iostream>

using namespace std;

class Printer; // 预先声明

class Building {
    private:
        int size;
        string year;

    private:
        void print_info() {
            cout << "The buliding is in " << size << " size,";
            cout << " it is " << year << " year" << endl;
        }

    public:
        Building(int s, string y) : size(s), year(y) {}
        friend class Printer; // 声明友元关系(若前面有预先声明, class可省略)
};

class Printer {
    public:
        void print_a_buliding(Building *temp) {
            temp->print_info();
        }
};

int main(void)
{
    Building *b = new Building(200, "2022");
    Printer myprinter;

    myprinter.print_a_buliding(b);

    delete b;

    return 0;
}
```

## 类内友元函数示例
```cpp
#include <iostream>
using namespace std;

class A; // 前向声明类 A

class B {
 public:
 void ask_for_number(const A &a); // 声明成员函数
};

class A {
 private:
 int number;

 public:
 A(int);

 friend void B::ask_for_number(const A& a); // 声明友元函数
};

// A 的成员实现
A::A(int a) : number(a) {}

// B 的成员函数实现
void B::ask_for_number(const A &a) {
 cout << "A's number: " << a.number << endl; // 通过友元函数访问
}

int main() {
 A a(12); // 创建 A 的对象
 B b; // 创建 B 的对象

 b.ask_for_number(a); // 调用 B 的成员函数访问 A 的私有成员

 return 0;
}

```
