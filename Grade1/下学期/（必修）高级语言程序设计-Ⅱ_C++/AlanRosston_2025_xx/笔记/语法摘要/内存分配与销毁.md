## 分配内存

### New 运算符的作用
`new` 运算符用于在堆（heap）上动态分配内存，并返回指向该内存的指针。与栈上的自动变量不同，用 `new` 分配的内存需要手动释放。

### 基本语法
```cpp
数据类型* 指针名 = new 数据类型;
数据类型* 指针名 = new 数据类型(初始值);
数据类型* 指针名 = new 数据类型[数组大小];
```

### 示例代码

#### 1. 分配单个变量的内存
```cpp
#include <iostream>
using namespace std;

int main() {
 // 分配一个int类型的内存
 int* ptr1 = new int; // 未初始化
 int* ptr2 = new int(42); // 初始化为42

 cout << "ptr1指向的值: " << *ptr1 << endl; // 未定义的值
 cout << "ptr2指向的值: " << *ptr2 << endl; // 42

 // 可以修改指向的值
 *ptr1 = 100;
 cout << "修改后ptr1指向的值: " << *ptr1 << endl; // 100

 return 0;
}
```

#### 2. 分配对象的内存
```cpp
#include <iostream>
using namespace std;

class Student {
public:
 string name;
 int age;

 Student() {
 name = "Unknown";
 age = 0;
 cout << "Student默认构造函数被调用" << endl;
 }

 Student(string n, int a) {
 name = n;
 age = a;
 cout << "Student带参构造函数被调用" << endl;
 }
};

int main() {
 // 分配Student对象的内存
 Student* stu1 = new Student(); // 调用默认构造函数
 Student* stu2 = new Student("张三", 20); // 调用带参构造函数

 cout << "学生1: " << stu1->name << ", " << stu1->age << endl;
 cout << "学生2: " << stu2->name << ", " << stu2->age << endl;

 return 0;
}
```

#### 3. 分配数组的内存
```cpp
#include <iostream>
using namespace std;

int main() {
 // 分配整型数组
 int size = 5;
 int* arr = new int[size];

 // 初始化数组
 for (int i = 0; i < size; i++) {
 arr[i] = i * 10;
 }

 // 输出数组内容
 cout << "数组内容: ";
 for (int i = 0; i < size; i++) {
 cout << arr[i] << " ";
 }
 cout << endl;

 // 分配字符数组
 char* str = new char[20];
 strcpy(str, "Hello World");
 cout << "字符串: " << str << endl;

 return 0;
}
```

## 销毁内存

### Delete 运算符的作用
`delete` 运算符用于释放之前用 `new` 分配的内存。不释放内存会导致内存泄漏，这是程序中的严重问题。

### 基本语法
```cpp
delete 指针名; // 释放单个对象的内存
delete[] 指针名; // 释放数组的内存
```

### 示例代码

#### 1. 释放单个变量的内存
```cpp
#include <iostream>
using namespace std;

int main() {
 // 分配内存
 int* ptr = new int(100);
 cout << "分配的值: " << *ptr << endl;

 // 释放内存
 delete ptr;

 // 良好习惯：将指针设为nullptr
 ptr = nullptr;

 cout << "内存已释放" << endl;

 return 0;
}
```

#### 2. 释放对象的内存
```cpp
#include <iostream>
using namespace std;

class Student {
public:
 string name;
 int age;

 Student(string n, int a) : name(n), age(a) {
 cout << "构造函数: 创建学生 " << name << endl;
 }

 ~Student() {
 cout << "析构函数: 销毁学生 " << name << endl;
 }
};

int main() {
 // 分配对象内存
 Student* stu = new Student("李四", 22);

 cout << "学生信息: " << stu->name << ", " << stu->age << endl;

 // 释放对象内存（会自动调用析构函数）
 delete stu;
 stu = nullptr;

 return 0;
}
```

#### 3. 释放数组的内存
```cpp
#include <iostream>
using namespace std;

int main() {
 // 分配数组内存
 int* arr = new int[5];
 char* str = new char[20];

 // 使用数组
 for (int i = 0; i < 5; i++) {
 arr[i] = i + 1;
 }
 strcpy(str, "Hello");

 cout << "数组: ";
 for (int i = 0; i < 5; i++) {
 cout << arr[i] << " ";
 }
 cout << endl;
 cout << "字符串: " << str << endl;

 // 释放数组内存（注意使用delete[]）
 delete[] arr;
 delete[] str;

 // 设为nullptr
 arr = nullptr;
 str = nullptr;

 cout << "数组内存已释放" << endl;

 return 0;
}
```

### 重要注意事项

#### 1. 配对使用
```cpp
int* ptr = new int(10); // 用new分配
delete ptr; // 必须用delete释放

int* arr = new int[5]; // 用new[]分配数组
delete[] arr; // 必须用delete[]释放
```

#### 2. 避免重复释放
```cpp
int* ptr = new int(10);
delete ptr;
// delete ptr; // 错误！重复释放会导致程序崩溃

ptr = nullptr; // 设为nullptr后可以安全地多次delete
delete ptr; // 对nullptr执行delete是安全的
```

#### 3. 完整的内存管理示例
```cpp
#include <iostream>
using namespace std;

int main() {
 cout << "=== 内存分配与释放示例 ===" << endl;

 // 1. 分配和释放单个整数
 int* num = new int(42);
 cout << "整数值: " << *num << endl;
 delete num;
 num = nullptr;

 // 2. 分配和释放数组
 int size = 3;
 double* scores = new double[size];
 scores[0] = 85.5;
 scores[1] = 92.0;
 scores[2] = 78.5;

 cout << "成绩: ";
 for (int i = 0; i < size; i++) {
 cout << scores[i] << " ";
 }
 cout << endl;

 delete[] scores;
 scores = nullptr;

 cout << "所有内存已正确释放" << endl;

 return 0;
}
```

### 总结
- **new**: 在堆上分配内存，返回指针
- **delete**: 释放 new 分配的单个对象内存
- **delete[]**: 释放 new[]分配的数组内存
- **重要原则**: 每个 new 都必须对应一个 delete，每个 new[]都必须对应一个 delete[]
- **良好习惯**: 释放后将指针设为 nullptr，避免悬空指针
