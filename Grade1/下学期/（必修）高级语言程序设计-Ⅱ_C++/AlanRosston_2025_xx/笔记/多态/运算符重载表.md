| **运算符** | **类型** | **是否必须为成员函数** | **声明方式（成员函数/友元函数）** |
| -------- | ------- | ------------- | ------------------------------------------------------------------------------------------------------------------- |
| `+` | 双目运算符 | 否 | `Vec operator+(const Vec& rhs) const;`（成员函数） <br>`friend Vec operator+(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `-` | 双目运算符 | 否 | `Vec operator-(const Vec& rhs) const;`（成员函数） <br>`friend Vec operator-(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `*` | 双目运算符 | 否 | `Vec operator*(const Vec& rhs) const;`（成员函数） <br>`friend Vec operator*(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `/` | 双目运算符 | 否 | `Vec operator/(const Vec& rhs) const;`（成员函数） <br>`friend Vec operator/(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `%` | 双目运算符 | 否 | `Vec operator%(const Vec& rhs) const;`（成员函数） <br>`friend Vec operator%(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `+=` | 复合赋值运算符 | 是 | `Vec& operator+=(const Vec& rhs);` |
| `-=` | 复合赋值运算符 | 是 | `Vec& operator-=(const Vec& rhs);` |
| `*=` | 复合赋值运算符 | 是 | `Vec& operator*=(const Vec& rhs);` |
| `/=` | 复合赋值运算符 | 是 | `Vec& operator/=(const Vec& rhs);` |
| `%=` | 复合赋值运算符 | 是 | `Vec& operator%=(const Vec& rhs);` |
| `==` | 比较运算符 | 否 | `bool operator==(const Vec& rhs) const;`（成员函数） <br>`friend bool operator==(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `!=` | 比较运算符 | 否 | `bool operator!=(const Vec& rhs) const;`（成员函数） <br>`friend bool operator!=(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `<` | 比较运算符 | 否 | `bool operator<(const Vec& rhs) const;`（成员函数） <br>`friend bool operator<(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `>` | 比较运算符 | 否 | `bool operator>(const Vec& rhs) const;`（成员函数） <br>`friend bool operator>(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `<=` | 比较运算符 | 否 | `bool operator<=(const Vec& rhs) const;`（成员函数） <br>`friend bool operator<=(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `>=` | 比较运算符 | 否 | `bool operator>=(const Vec& rhs) const;`（成员函数） <br>`friend bool operator>=(const Vec& lhs, const Vec& rhs);`（友元函数） |
| `!` | 单目运算符 | 否 | `bool operator!() const;`（成员函数） <br>`friend bool operator!(const Vec& v);`（友元函数） |
| `~` | 单目运算符 | 否 | `Vec operator~() const;`（成员函数） <br>`friend Vec operator~(const Vec& v);`（友元函数） |
| `++`（前置） | 单目运算符 | 是 | `Vec& operator++();` |
| `++`（后置） | 单目运算符 | 是 | `Vec operator++(int);` |
| `--`（前置） | 单目运算符 | 是 | `Vec& operator--();` |
| `--`（后置） | 单目运算符 | 是 | `Vec operator--(int);` |
| `=` | 赋值运算符 | 是 | `Vec& operator=(const Vec& rhs);` |
| `[]` | 下标运算符 | 是 | `int& operator[](size_t index);` |
| `()` | 函数调用运算符 | 是 | `int operator()(int x, int y);` |
| `->` | 指针访问运算符 | 是 | `Vec* operator->();` |
| `<<` | 输入输出运算符 | 否 | `friend ostream& operator<<(ostream& os, const Vec& v);` |
| `>>` | 输入输出运算符 | 否 | `friend istream& operator>>(istream& is, Vec& v);` |
| `&` | 单目运算符 | 否 | `Vec* operator&();` |
| `*` | 单目运算符 | 否 | `int operator*() const;`（成员函数） |
| `new` | 内存分配运算符 | 是 | `void* operator new(size_t size);` |
| `delete` | 内存释放运算符 | 是 | `void operator delete(void* ptr);` |
