## 运算符重载规则

基本规则：
1. 运算对象**至少包含一个**自定义数据类型，即不能重载标准类型的运算符
2. 不能改变运算符的**优先级**、**结合性**和**操作数个数**
3. 只重载预定义运算符，不能自创

关于**成员**和**非成员**运算符：
1. 必须重载为成员函数的运算符
	1. 赋值 `=`
	2. 下标 `[]`
	3. 函数调用 `()`
	4. 成员访问 `->`
2. 通常应该作为非成员函数，且推荐作为友元函数的运算符
	1. 所有**二元运算符**
	2. 输入输出运算符 `<<` 和 `>>`
3. 不可重载
	1. `sizeof`
	2. `.`
	3. `.*`
	4. `::`
	5. `?:`

约定俗成：
1. 不改变对象状态，就应该用 `const` 修饰成员函数
2. 保持原始语义

特别注意：
1. 用**友元函数**实现的运算符重载，在双目运算符中可以实现两个类对调位置也能执行；但是用**成员函数**实现的运算符重载，隐含了 `this` 指针，因此对于双目运算符只能由一个其他参数，且必须这个参数在使用中必须放在运算符右边

## 运算符重载举例

### 双目运算符 `+` 的友元函数实现
声明一个类 `Vec`，用于表示向量：
```cpp
class Vec {
    private:
        vector<int> values;

    public:
        Vec() {}
        Vec(vector<int> v) : values(v) {}

}
```

我们在 `public` 内 (实际上哪里都可以) 添加声明：
```cpp
friend Vec operator+ (const Vec &l, const Vec &r);
```
表示一个友元函数作为双目运算符 `+`，然后我们在类外实现它 (也可以实现在内部)：
```cpp
Vec operator+ (const Vec &l, const Vec &r) {
    if(l.values.size() != r.values.size()) {
        throw invalid_argument("Vectors must have the same size.");

    }

    vector<int> res(l.values.size());
    for(int i = 0; i < l.values.size(); i++) {
        res[i] = l.values[i] + r.values[i];
    }

    return Vec(res);
}
```

### 单目运算符 `[]` 的成员函数实现

直接在 `public` 中添加成员函数：
```cpp
        int operator[] (size_t index) {
            if(index >= values.size()) {
                cout << "Out of Range" << endl;
            } else {
                return values[index];
            }
        }
```
即可实现 `[]` 运算符
> 既然是成员函数
